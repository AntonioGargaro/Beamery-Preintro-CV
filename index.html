PROJECTS

Dissertation

My dissertation stemmed from my passion of cryptocurrnecies and trading them on exchanges. I wanted to develop a tool that would make it easier to understand the direction a market was heading in using technical analysis. This required an understanding of how financial markets worked and what similiarties were present in the crypto market.

The project was developed as a web app that used Software-as-a-Service (SaaS) to perform the technical analysis on the server and forward the output to a UI based dashboard in the browser. From knowing virtually nothing about developing a web app, I researched and developed this project using Python's Flask for the server and React for the UI.
